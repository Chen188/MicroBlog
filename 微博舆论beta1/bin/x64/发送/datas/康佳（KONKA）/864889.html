按键说明

家长模式
右侧上下方向键：切换菜单，滚动浏览文字，通话中调节音量。呼叫时可切换数字。
呼叫键（接听键，确认键）：
①在家长模式下待机时按此键快捷进入孩子模式;在孩子模式下按键进入电话簿。
②来电话时为“接听”键。
③“拨号”键
④短信编辑完毕后的“发送”键。
⑤进入菜单的“确认”键。
挂机键（取消键）：
①长按开机/关机键。
②据绝来电/结束通话键。
③某些情况下的取消/退出键。SOS帮助键（紧急键）：
①部分功能菜单界面，短按将显示帮助信息。
②在待机下长按3秒左右会依次自动紧急拨打热键1/2/3/4号码，直至呼叫成功为止。
③输入好友电话或输入密码界面，短按SOS键为确认选择的数字。

数字热键（1—4）：
①待机状态，短按分别呼叫[我的好友]中对应的1—4的好友；
②在“我的好友”、“我的短消息”、“通话记录”中，
1键〈——〉用于删除一条记录；
2键〈——〉用于删除全部记录；在我的好友中，
3键〈——〉用于增加一条记录；
4键〈——〉用于修改一条记录。在“呼叫’、”设置好友电话”、”设置监护人”等涉及编辑数字时，
1键〈——〉用于删除最后一位数字，
2键〈——〉用于删除全部数字。
③待机情况下，长按3键上锁，长按4键解锁。
我的好友

操作方法:
按侧键 >>进入我的好友
按通话键 >> 进入按“3”增加好友号码
按上下侧键>>移动所需号码
按SOS确认>>按通话键>>添加成功
注:管理好友号码.
按1键删除单个好友号码；
按2键删除全部好友号码；
另家长可通过发送短信控制格式（好友，编号，号码）
续增加好友号码。
我的短消息

1.写短信
进入菜单>>我的短消息>>写短信>>发给谁>>进入好友名单>>选择好友 >>要发什么>>常用短语>>选择短语>>发送
2.我收到的
进入我收到的>>按通话查看>>按侧健下查看下一条>>按通话键查看短信内容>>按侧下键往下查看>>按挂机键返回
3.常用短语
进入常用短语>>按侧键查看>>按挂机键返回
短信管理:
按1键删除一条信息
按2键删除全部信息
我的通话记录

1、未接电话
进入菜单 >> 按通话查看通话记录
按侧键上下翻看记录>> 按挂机键返回
2、已拨电话 方法同1
3、已接电话 方法同1
管理通话记录:
按1键删除一条通话记录
按2键删除全部通话记录
呼叫

按通话键进入菜单>>上下侧健移动所要拨打号码上
按SOS键确认号码>> 选好号后按通话键进行通话
按挂机键结束通话
通话过程中,按通话键可启用免提
在呼叫选择号码时,输入错误可按1键删除.按2键删除全部
此项只限于家长模式
家长设置

1.语言设置
进入菜单后 >> 按侧键上下移动 >> 按通话键选择所需要语言（中文/英文）
2.设置时间
A.修改时间
进入菜单后 >> 修改时间按通话键切换小时和分钟>> 按侧键上下调整时间>> 调整好小时按返回 >> 按通话键保存设置
B.时间格式
进入后按侧健上下选择>> 按通话键确认
3.设置日期 同设置时间操作一样
4.限制通话时长
进入菜单>> 按侧键上下选择所需 >> 按通话键确认
5.定时关机
A.状态
进入菜单 >> 按侧键选择开/关 >> 按通话键确认
B.关机时间
进入菜单 >> 按侧键调整好时间 >> 按挂机键退出 >> 按通话键保存设置
C.开机时间 设置方法同定时关机
6.修改密码
进入菜单 >> 先输入原始密码 >> 按通话确认 >> 再次输入新的密码 >> 按通话键确认
7.短信服务中心
使用新SIM卡时，若遇到不能正常发送短信情况，可能是SIM卡邦定的短信中心号码不正确，可在“短信服务中心”功能下重新设置，要输入的短信中心号码可以咨询当地网络运营商。
8.待机界面显示
待机界面显示开启的状态下，监护人可以通过手机发送“孩子姓名”作为孩子手机的待机图片。
9.恢复出厂设置
进入“恢复出厂设置”输入正确的密码并确定后，本机所有的数据均被清除，所有的参数恢复为出厂与设置，密码也将恢复为“0000”
注意：
1.限制通话时长只限于孩子模式；
2.恢复出厂设置后，所有用户设置数据都将丢失，请慎重使用。
